Walmart Data Analysis: End-to-End SQL + Python Project
Overview

This project delivers a complete data analysis workflow using Walmart sales data. It covers data ingestion, cleaning, transformation, SQL-based analytics, and structured documentation. Python is used for ETL and preprocessing, while MySQL and PostgreSQL are used for executing complex SQL queries to extract business insights.

Project Workflow

Environment setup (Python, VS Code, MySQL, PostgreSQL)

Kaggle API configuration and dataset download

Library installation and data loading

Exploratory data analysis

Data cleaning and type standardization

Feature engineering (e.g., Total_Amount calculation)

Loading cleaned data into SQL databases

Advanced SQL analysis to answer business questions

Documentation and project publishing

Key Skills Demonstrated

Data cleaning and preprocessing in Python

SQL query building (joins, window functions, aggregations)

ETL workflow creation using SQLAlchemy

Multi-database integration (MySQL and PostgreSQL)

End-to-end project structuring and documentation
